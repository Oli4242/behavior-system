# Behavior System

*A behavior system for javascript games inspired by Construct 2.*

[![js-standard-style](https://cdn.rawgit.com/feross/standard/master/badge.svg)](https://github.com/feross/standard)

## Install

`npm install behavior-system --save`

or

```html

```
